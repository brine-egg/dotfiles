# brine's personal dotfiles and system config

> [!IMPORTANT] NixOS and nix-darwin config are **UNTESTED** and contain only placeholder stubs for now.

## About

This is a repo for version-controlling and syncing config files across my personal devices, meaning it is highly
personalised (and sloppily vibe-coded). This is **NOT** meant to be a template for your own config, but feel free to
clone the repo and modify it for whatever use-cases you have.

If you need actual templates for your own Nix config, here are some good options:

- [dustinlyons/nixos-config](https://github.com/dustinlyons/nixos-config)

- [Misterio77/nix-starter-configs](https://github.com/Misterio77/nix-starter-configs)

## Structure

This flake manages three kinds of configuration:

- **Home Manager** (standalone) — user-level dotfiles, shell, and programs.
- **NixOS** — Linux system configuration.
- **nix-darwin** — macOS system configuration.

```
.
├── flake.nix                 # all flake outputs
├── home/                     # Home Manager config
│   ├── home.nix
│   ├── config/               # shared, linux, darwin HM modules
│   └── packages/              # shared, linux, darwin packages
├── nixos/                    # shared NixOS system modules
│   ├── config/
│   │   └── system/           # one file per system component (boot, networking, users, …)
│   └── packages/
├── darwin/                   # shared nix-darwin system modules
│   └── config/
└── hosts/                    # per-host config
    ├── desktop/               # NixOS host
    ├── laptop/                # NixOS host
    └── macbook/               # nix-darwin host
```

## Setup

### Clone repo

```bash
git clone https://github.com/brine-egg/dotfiles && cd dotfiles
```

### Non-NixOS Linux (Home Manager only)

Nix cannot apply the system-level config on non-NixOS distros. Use standalone Home Manager:

#### 1. Bootstrap Home Manager

```bash
nix run home-manager/master -- init --switch
```

#### 2. Switch home config

```bash
home-manager switch --flake .#brine
```

### NixOS (+ standalone Home Manager)

Uses standalone Home Manager rather than a NixOS module, system and home config are applied separately.

#### 1. Write host config

Declare your actual hostname using `nixosConfigurations` in flake.nix. Write your device-specific config to
`hosts/your-hostname`. Make sure to copy the hardware configuration generated by the NixOS installer
(`/etc/nixos/hardware-configuration.nix`) into your host config.

#### 2. Switch system config

```bash
sudo nixos-rebuild switch --flake .#desktop
# Replace `desktop` with your actual hostname
```

#### 3. Bootstrap Home Manager

```bash
nix run home-manager/master -- init --switch
```

#### 4. Switch home config

```bash
home-manager switch --flake .#brine
```

### macOS (nix-darwin + standalone Home Manager)

macOS system config is managed by nix-darwin, home config is managed by standalone Home Manager. Both are applied
separately.

#### 1. Write host config

Declare your actual hostname using `darwinConfigurations` in `flake.nix`. Write your device-specific config to
`hosts/your-hostname`.

#### 2. Bootstrap nix-darwin and switch system config

```bash
sudo nix run nix-darwin/master#darwin-rebuild -- switch --flake .#macbook
# Replace `macbook` with your actual hostname
```

#### 3. Bootstrap Home Manager

```bash
nix run home-manager/master -- init --switch
```

#### 4. Switch home config

```bash
home-manager switch --flake .#brine-darwin
```

GUI apps installed via Nix packages are automatically wrapped as `.app` bundles and indexed by Spotlight, powered by
[mac-app-util](https://github.com/hraban/mac-app-util). No Homebrew needed.

## General Usage (post-setup)

### With helper utility

The [NH helper utility](https://github.com/nix-community/nh) is included as package, it can be used to switch config,
rollback or perform garbage collection with more readable output.

#### Switch NixOS config

```bash
nh os switch . -H your-hostname
```

#### Switch macOS config

```bash
nh darwin switch . -H your-hostname
```

#### Switch home config

```bash
nh home switch . -c config-name
```

### Without helper utility

#### Switch NixOS config

```bash
sudo nixos-rebuild switch --flake .#your-hostname
```

#### Switch macOS config

```bash
sudo darwin-rebuild switch --flake .#your-hostname
```

#### Switch home config

```bash
home-manager switch --flake .#config-name
```

### Maintenance

#### Update flake.lock

```bash
nix flake update --flake .
```

#### Manually run garbage collection

```bash
sudo nix-collect-garbage
```

```bash
# If you wanna use helper:
nh clean all
```
